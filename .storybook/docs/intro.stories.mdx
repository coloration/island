# @coloration/island

a vue ui component library



## Startup

``` bash
$ npm install @coloration/island
```

## Usage

``` html
<script lang="ts" setup>
import { IButton } from '@coloration/island'
</script>

<template>
  <i-button>Greeting</i-button>
<template>
```

- [Document](https://coloration.github.io/island)
