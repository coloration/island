# Vue Component Library Boilerplate

## 🦄 Usage

``` bash
$ npm run dev

$ npm run build:book
$ npm run build
```

## 🚀 Features

- [x] [VueDemi](https://github.com/vueuse/vue-demi) for Vue2 & Vue3
- [x] [Windi CSS](https://github.com/windicss/windicss)
- [x] [Storybook](https://github.com/storybookjs/storybook)
- [ ] Unit Test
